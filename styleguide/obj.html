List Grid (.o-ListGrid)

A List Grid is simply a list of items arranged like blocks in a grid. It has the structure of a list in that all items share the same container (unlike tables where items are in different rows), while at the same time appearing like a table with rows and columns. The List Grid is flexible and responsive, adjusting the number of columns based on screen size.

The container must have the .o-ListGrid class and each item must have the .o-ListGrid__Item class.
